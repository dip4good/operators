ARITHMETIC OPERATORS

Arithmetic Operation operates on two numbers. They are:

1. Addition

The addition operator add numbers

2. Subtraction

The subtraction operator subtract numbers

3. Multiplication

The multiplication operator multiplies numbers

4. Modulus

The modulus operator leaves the remainder of numbers

5. Division

The division operator divides numbers

6. Exponential

The exponential operator raises numbers to power

COMPARISON OPERATORS

Comparison operators are use to test for true or false. Examples are:
1. Equals to == e.g y==8

2. Equals value and Equal type === e.g w===9

3. Not Equals to != e.g n!=4

4. Not Equals value and Equal type !== e.g z!==3

5. Greater than > e.g t>2

6. Less than < e.g r<8

7. Greater than or Equals to >= e.g d>2

8. Less than < e.g b<=8
